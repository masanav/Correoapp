La aplicación

Permite enviar mensajes de correo a múltiples destinatarios, como una herramienta de email

Con backend de Spring

Microservicios

  • Envío de mensajes en multihilo
  • Autenticación en Oauth2
  • Endpoints asegurados por roles
  • Credenciales aisladas del resto de la aplicación
  • Configuración centralizada
  • Diferentes versiones de Spring Boot entre microservicios
  • Documentación del backend centralizada
  • Diferentes proveedores de bases de datos SQL
  • Entidades, servicios, controladores rest
  • Interfaces, jpa
  • Documentación, configuraciones personalizadas, multihilo, asíncronismo
